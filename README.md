## Setup

To run this project, install it locally using npm:

```
From client terminal
1. cd bcs-ecommerce-project
2. cd client
3. npm install
3. npm start
```

```
From server terminal
1. cd bcs-ecommerce-project
2. cd server
3. npm install
3. nodemon
```
